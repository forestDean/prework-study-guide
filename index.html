HTML

  • The <!DOCTYPE html> declaration tells the browser what document type it should expect. It is case-insensitive.
  • The <html> element represents the root/start of our HTML document and can contain language attributes.
  • The <head> element contains information about the webpage in metadata tags/elements.
  • The <body> element represents the visible content shown to the user.
  • The <title> element defines the title for a webpage. It is required and important for SEO.
  • The <style> element defines the CSS styles associated with a website.
  • The <base> element defines the base URL for a webpage.
  • The <link> element connects an external resource to the HTML document.
  • The <style> element defines metadata such as the character set, description, keywords, author, and viewport.
  • The <script> element embeds client-side scripts in the HTML document.
  • The <div> element is non-semantic and used to create a division in the page.
  • The <div> element is non-semantic and used to create an inline division.
  • The <h1> <h2> <h3> <h4> <h5> <h6> elements represent the level of heading a text block represents.
  • The <p> element represents a paragraph or block of text.
  • The <ul> element represents an unordered list.
  • The <ol> element represents an ordered list.
  • The <li> element represents a list item.
  • The <img> element contains information about images on the webpage. It contains various attributes.
  • The <br> element creates a line break of empty space.
  • The <a> element creates links to the same webpage or other webpages. The href attribute points to the URL for the link.

Git

  • git status: checks what branch we are currently on
  • git pull origin main: pulls down the latest main branch
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stages all files in the working branch.
  • git commit -m "short message": commits staged files to our repository, with a message.
  • git push origin branch-name: pushes 'branch-name' changes to the GitHub repository.
